Class Product Proposal - DinoRush

Product Name

DinoRush

Application Overview

DinoRush is a user-friendly mobile and web application designed for individuals to have fun and play the game online. Users can log in daily, view their results, and spend a good time playing.

Technology Overview

Languages: HTML, CSS, and JavaScript

Libraries: D3.js

Platforms: Web and Mobile (iOS and Android) and computers

Hosting: Google Chrome

Proposal Last Updated

December 8, 2024

Feature List
