Class Product Proposal - Would You Rather?

Product Name

Would You Rather?

Application Overview

WYR? is a user-friendly mobile and web application designed for individuals to have fun and play the game online. Users can log in daily, view their results, and receive personalized funny feedback.

Technology Overview

Languages: Python, JavaScript

Libraries: D3.js

Platforms: Web and Mobile (iOS and Android)

Hosting: Heroku for backend, Netlify for frontend deployment

Proposal Last Updated

November 4, 2024

Feature List

  1. User authentication
  2. Activity logging
  3. Choosing either or
  4. Personalized funny feedback

Future Feature List

  • Integration with smart home devices
  • Gamification elements
  • Enhanced data visualization features
  • Multi-language support

Basic Technical Features

  • Giving feedabck
  • RESTful API for data handling
  • Responsive design with React
  • Data persistence using PostgreSQL

Advanced Technical Features

  • Real-time data visualization with D3.js
  • Implementation of a recommendation engine
  • CI/CD pipelines for automated testing and deployment
